# PRD Step 2 — 레시피 생성

## 개요

1단계에서 추출된 재료 목록을 `openai/gpt-oss-120b:free` 모델에 전달하여 실용적인 레시피를 생성하고 화면에 표시한다.

---

## 목표

- 냉장고에 있는 재료만으로 만들 수 있는 레시피를 복수로 제안한다.
- 사용자의 조리 시간, 인분 수, 식단 제한을 반영한 맞춤형 레시피를 생성한다.
- 3단계(레시피 저장)로 연결될 수 있는 구조화된 데이터로 결과를 반환한다.

---

## 사용자 스토리

| # | As a… | I want to… | So that… |
|---|-------|-----------|----------|
| 1 | 사용자 | 냉장고 재료로 만들 수 있는 레시피를 추천받고 싶다 | 뭘 요리할지 고민하지 않아도 된다 |
| 2 | 사용자 | 조리 시간과 인분 수를 설정하고 싶다 | 나의 상황에 맞는 레시피를 받을 수 있다 |
| 3 | 사용자 | 레시피의 조리 과정을 단계별로 보고 싶다 | 쉽게 따라할 수 있다 |
| 4 | 사용자 | 마음에 드는 레시피를 저장하고 싶다 | 나중에 다시 볼 수 있다 |

---

## 기능 요구사항

### F1. 레시피 생성 옵션 설정
- 조리 시간: 15분 이하 / 30분 이하 / 60분 이하 / 제한 없음
- 인분 수: 1인분 / 2인분 / 4인분 이상
- 식단 제한 (다중 선택): 없음 / 채식 / 비건 / 글루텐프리 / 유제품프리

### F2. 레시피 생성 API 호출
- 모델: `openai/gpt-oss-120b:free`
- 1단계 재료 목록 + 사용자 설정을 프롬프트로 전달
- 최대 3개의 레시피 제안

**시스템 프롬프트 (예시)**
```
You are a professional recipe generator.
Given a list of available ingredients, generate practical Korean-style recipes.
Always respond in Korean. Return valid JSON only.
```

**사용자 프롬프트 (예시)**
```
사용 가능한 재료: 달걀, 우유, 당근, 양파, 간장
조리 시간: 30분 이하
인분 수: 2인분
식단 제한: 없음

위 재료로 만들 수 있는 레시피 3가지를 JSON으로 생성해줘.
```

**응답 형식 (JSON)**
```json
{
  "recipes": [
    {
      "id": "recipe_001",
      "name": "달걀 채소볶음",
      "description": "냉장고 재료로 만드는 간단한 영양 볶음",
      "cook_time_minutes": 20,
      "servings": 2,
      "difficulty": "쉬움",
      "ingredients": [
        { "name": "달걀", "amount": "3개" },
        { "name": "당근", "amount": "1/2개" }
      ],
      "steps": [
        { "step": 1, "description": "당근을 채 썬다." },
        { "step": 2, "description": "달걀을 풀어 소금으로 간한다." }
      ],
      "tips": "달걀을 넣기 전 팬을 충분히 달구면 더 맛있어요.",
      "nutrition": {
        "calories": 320,
        "protein": "18g",
        "carbs": "12g",
        "fat": "22g"
      }
    }
  ]
}
```

### F3. 레시피 결과 표시
- 카드형 UI로 3개의 레시피 표시
- 각 카드: 레시피명, 조리 시간, 난이도, 간단 설명
- 카드 클릭 시 상세 페이지(재료 목록 + 단계별 조리법)로 이동
- 상세 페이지에서 "저장하기" 버튼으로 3단계 연계

### F4. 레시피 재생성
- "다른 레시피 보기" 버튼으로 새로운 레시피 재요청
- 동일 재료로 다른 조합 생성

---

## API 엔드포인트

### `POST /api/recipes`

**Request**
```json
{
  "ingredients": ["달걀", "우유", "당근", "양파"],
  "options": {
    "max_cook_time": 30,
    "servings": 2,
    "dietary": []
  }
}
```

**Response (200)**
```json
{
  "success": true,
  "recipes": [ ... ]
}
```

### `GET /api/recipes/stream` (SSE)

스트리밍 응답 지원 — 레시피가 생성되는 대로 클라이언트에 전송.
